# 🎬 Movie App (React Project)

## 📌 Overview

This is a simple Movie App built using React. The application allows users to search for movies, view results, and add movies to their favorites list.

This project was created by following a YouTube tutorial

---

## 🚀 Features

* 🔍 Search movies using an API
* 🎥 Display movie posters and details
* ❤️ Add and remove favorite movies
* 💾 Save favorites using local storage
* ⚡ Fast and responsive UI

---

## 🛠️ Technologies Used

* React JS
* JavaScript (ES6+)
* CSS
* React Router
* Movie API (e.g., OMDb API)

---

## 📂 Project Structure

* `components/` → Reusable UI components
* `pages/` → Main pages (Home, Favorites)
* `App.jsx` → Main app routing
* `NavBar.jsx` → Navigation bar

---

## ▶️ How to Run the Project

```bash
npm install
npm start
```
